# Finance Dataset Analysis

## Project Overview
This repository contains a comprehensive data analysis project focused on financial performance metrics. The objective was to extract actionable business insights from a dataset of 8500 records, covering diverse financial activities. This project demonstrates proficiency in data cleaning, SQL query optimization, Python-based statistical analysis, and interactive dashboard creation.

## Key Features
- **Data Cleaning & Preprocessing:** Utilized Python (Pandas) to handle missing values, format data types, and normalize structures for analysis.
- **SQL Analysis:** Implemented complex SQL queries to aggregate financial metrics and segment customer data.
- **Interactive Dashboard:** Developed a Power BI dashboard to visualize key performance indicators (KPIs) and track financial trends.
- **Business Reporting:** Compiled findings into a strategic professional report.

## Tech Stack
- **Languages:** Python (Pandas, Matplotlib, NumPy), SQL
- **Visualization:** Power BI
- **Tools:** Jupyter Notebook, Git/GitHub

## Project Structure
- `Finance Project.ipynb`: Python notebook containing the end-to-end data processing workflow.
- `Finance_Analysis_Final.sql`: SQL scripts used for data extraction and transformation.
- `Finance Analysis Dasshboard.pbix`: Power BI project file for data visualization.
- `Finance_Dataset_Analysis_Final_Report.pdf`: Final strategic business report.
- `Professional_Finance_Project_Dataset_v2.csv`: The primary dataset analyzed.

## Insights
Total Financial Performance: Analyzed a total revenue of $21.42 Million, generating a total profit of $4.79 Million.
Top-Performing Segment: The Consumer segment emerged as the primary revenue driver, accounting for 48.96% ($10.49M) of total sales.
Operational Efficiency: The business maintains a healthy average profit margin of 22.45% across all transactions.
Transaction Profile: The average transaction value stands at $2,598.45, indicating a consistent high-value customer base.
